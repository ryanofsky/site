Rvalue references for G++

Hi, this is a little project page for my attempt to add rvalue references to G++. My current implementation includes all the language changes described in the rvalue reference proposals except for automatic cast to rvalue in return statements (which is just syntactic sugar that doesn't prevent rvalue references from providing move semantics or perfect forwarding.)

I need to do more testing and hash out some issues with the proposal on comp.std.c++, but eventually, I should have a complete patch that I'll submit for inclusion in GCC. For now, I'm working off a slightly out of date mirror of the GCC HEAD sources and tracking my changes in a local Subversion repository. The address of the repository and a copy of the latest diff are available below.

Russell Yanofsky
russ@yanofsky.org

Download

My changes are available from a public Subversion repository at http://russ.yanofsky.org/svn/rref/

If you don't have Subversion, you can see a cached copy of the latest diff, see a summary of code changes, or browse the repository root.

If you do have Subversion, you can check out the sources with

svn co http://russ.yanofsky.org/svn/rref/trunk/ rref

or get the latest diff with

svn diff http://russ.yanofsky.org/svn/rref/vendor/gcc/current http://russ.yanofsky.org/svn/rref/trunk
