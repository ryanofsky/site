Rvalue references for G++

Hi, this is a little project page for my attempt to add rvalue references to G++. The current implementation includes all the language changes described in the rvalue reference proposals and is being submitted for inclusion in GCC.

I received help from a number of people while implementing these changes. Pedro Lamarão assisted with testing and implemented support for GCC's --std=c++0x option. Doug Gregor added support for automatic rvalue casting in return and throw statements. Howard Hinnant provided a ton of help and support by email, and also wrote a whole suite of unit tests which are included in the GCC patch.

Russell Yanofsky
russ@yanofsky.org

Download

The project's changes are available from a public Subversion repository which mirrors the GCC trunk. The repository address is:

http://russ.yanofsky.org/svn/rref/

If you don't have Subversion, you can see a cached copy of the latest diff, see a summary of code changes, or browse the repository root.

If you do have Subversion, you can check out the sources with

svn co http://russ.yanofsky.org/svn/rref/trunk/ rref

or get the latest diff with

svn diff http://russ.yanofsky.org/svn/rref/vendor/gcc/current http://russ.yanofsky.org/svn/rref/trunk
